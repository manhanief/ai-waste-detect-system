# AI Waste Detect System

A desktop application that uses a **Convolutional Neural Network (CNN)** to classify images of recyclable waste into 5 material categories: **cardboard, glass, metal, paper, and plastic**.

Built with TensorFlow/Keras and a tkinter GUI. Includes a custom CNN trained from scratch, an optional MobileNetV2 transfer-learning model, evaluation tools, and a user-friendly desktop interface.

## Requirements

- **Python 3.11** (required — TensorFlow on macOS only supports 3.11)
- macOS (tested on Apple Silicon)

---

## Setup

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/ai-waste-detect-system.git
cd ai-waste-detect-system
```

### 2. Create a virtual environment

```bash
/opt/homebrew/bin/python3.11 -m venv .venv311
```

> If Homebrew Python 3.11 is not installed:
> ```bash
> brew install python@3.11
> ```

### 3. Install dependencies

```bash
.venv311/bin/python -m pip install --upgrade pip
.venv311/bin/python -m pip install -r requirements.txt
```

### 4. Install tkinter support (macOS only, one-time)

```bash
brew install python-tk@3.11
```

---

## Running the GUI (pre-trained model included)

A trained model (`custom_cnn.keras`) is included in the repo — no training needed to try the app.

```bash
bash run_gui.sh
```

The window will open. Click **Browse File**, select any image of waste material, and the AI will predict its category along with a confidence score and recycling tip.

## Training from scratch

### 1. Get the dataset

Download **TrashNet** from [garythung/trashnet](https://github.com/garythung/trashnet) (the `dataset-resized.zip` file), unzip it, and place the class folders inside `dataset-resized/`:

```
dataset-resized/
    cardboard/
    glass/
    metal/
    paper/
    plastic/
```

> The `trash/` folder from TrashNet is not used — this project focuses on 5 clean recyclable categories.

### 2. Split into train / test

```bash
.venv311/bin/python split_dataset.py
```

Creates `dataset/train/` (80%) and `dataset/test/` (20%).

### 3. Train

```bash
bash run_train.sh
```

This trains the custom CNN for up to 30 epochs with early stopping, saves the best model as `custom_cnn.keras`, and writes accuracy/loss curves to `curves_custom.png`.

To run the **transfer-learning** experiment (MobileNetV2), open `train.py`, change:
```python
MODEL_TYPE = "transfer"
```
then run `bash run_train.sh` again (saves `transfer_cnn.keras`).

### 4. Evaluate

```bash
.venv311/bin/python evaluate.py
```

Prints a per-class precision / recall / F1 report and saves a confusion matrix image.

### 5. Quick command-line prediction

```bash
.venv311/bin/python predict.py path/to/image.jpg
```

## File Overview

| File | Purpose |
|---|---|
| `gui.py` | Desktop GUI — browse an image and see the prediction |
| `train.py` | Train the CNN and save the model |
| `model.py` | Custom CNN and MobileNetV2 transfer-learning architectures |
| `data_loader.py` | Dataset pipeline, augmentation, and normalisation layers |
| `evaluate.py` | Confusion matrix and per-class metrics on the test set |
| `predict.py` | Classify a single image from the command line |
| `split_dataset.py` | Split raw images into `train/` and `test/` folders |
| `run_gui.sh` | Launch the GUI using the correct Python environment |
| `run_train.sh` | Train using the correct Python environment |
| `requirements.txt` | Python dependencies |
| `class_names.json` | Saved class label order (auto-generated by `train.py`) |
| `custom_cnn.keras` | Pre-trained model weights (ready to use) |

## Recommended Run Order (training from scratch)
